3. Sublime Text（可免费试用，无功能限制）
平台：全平台
特点：
- 极速启动，界面清爽
- “Goto Anything”（Ctrl+P）快速跳转
- 支持多光标、宏和正则
是否完全免费？
- 可无限期试用，偶尔弹出购买提示，但不影响使用。
官网：https://www.sublimetext.com
备注：✅ 很多人把它当“高级记事本”长期免费用。

二、专注 Markdown / 写作
4. Typora（旧版仍可用）
平台：全平台
特点：
- 所见即所得的 Markdown 编辑体验
现状：
- 新版已收费，0.11.18 旧版仍可免费下载使用（非官方更新停止）。
替代推荐：
- MarkText（完全开源免费）

5. MarkText
平台：Windows / macOS / Linux
特点：
- 开源免费，界面类似 Typora
- 实时预览、数学公式、代码块高亮
官网：https://marktext.app

三、极简 & 轻量级（适合快速记录）
6. Notepad（Windows 自带）
- 最基础，Win10/11 的新版已支持 Unix 换行符和基本编码，够用。

7. TextEdit（macOS 自带）
- 切换到“纯文本模式”（格式 → 纯文本）即可当轻量编辑器用。

8. QuickEdit（Android）
- 手机上编辑代码/文本神器，支持大文件、语法高亮。
- 可在 Google Play / 国内应用商店下载。

四、开发者专用（支持代码高亮、项目管理）
常见编辑器与特点：
- Vim / Neovim：终端神器，学习曲线陡，但效率极高（适合 Linux/macOS 用户）。
- Nano：终端新手友好，Ctrl+O 保存，Ctrl+X 退出。
- Joplin：开源笔记，支持 Markdown、端到端加密与多端同步（可替代印象笔记）。

推荐表（根据需求）：
- 写代码（前端 / Python / Java 等）：VS Code
- 快速编辑配置文件或日志（Windows）：Notepad++
- 极致速度 + 简洁界面：Sublime Text（免费试用）
- 写 Markdown 文档：MarkText 或 VS Code
- 手机上编辑文本：QuickEdit（安卓） / iA Writer（iOS）
- 做笔记 + 同步：Joplin（完全免费开源）

五、VS Code 的优势（补充）
平台：Windows / macOS / Linux
优点：
- 轻量且功能强大，启动快速，适合从简单编辑到大型项目的开发。
- 扩展生态丰富（Extensions Marketplace），支持各种语言、格式化器、调试器与工具。
- 内置终端、Git 集成和可视化调试器，减少切换、提高工作流效率。
- 智能补全（IntelliSense）、代码片段、重构和多光标编辑，显著提升编码速度。
- 支持远程开发（Remote - SSH / Containers / WSL），可在容器或远程主机上无缝工作。
- 配置高度可定制，支持设置同步、任务、工作区和键盘映射，便于团队协作与个人习惯。
- 免费且开源（核心），并提供企业/付费扩展（如 Codespaces、Copilot）满足高级需求。
